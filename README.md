# manhattan-weather-tracker
This project tracks Manhattan's temperatures 4 times a day. It was built as part of an Intro to Data Science class to fetch, analyze, and display data. To provide these temperatures, the script utilizes the Open-Meteo API to fetch historical temperature records from the previous 5 years alongside current 7-day weather forecasts. You can customize the variables to work for othe durations and destinations as well. Just go into the weather.py and edit some variables.
